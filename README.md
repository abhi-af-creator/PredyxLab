# PredyxLab

**PredyxLab** is a modular, API-driven stock prediction and analytics platform.

### Key Features
- Historical market data visualization
- ML-based price prediction (Open / Close)
- Clean API for frontend & trading bots
- Designed for extensibility and experimentation

### Tech Stack
- Backend: Python, FastAPI
- ML: scikit-learn (initial), deep learning later
- Frontend: React
- Data: yFinance (initial)

> ⚠️ Educational and research use only. Not financial advice.
